# **App Name**: SquadFlow

## Core Features:

- Player Profile Management: Team administrators can create, view, edit, and delete player profiles, including name, nickname, and preferred position.
- Game Schedule Management: Team administrators can add, edit, and view details for upcoming games, specifying date, time, location, and type (e.g., training, league game, friendly).
- Player Attendance Confirmation: A dedicated page where players can log in, view scheduled games, and confirm or decline their attendance for each upcoming game.
- Team Attendance Dashboard: A dashboard for team management to easily visualize attendance for upcoming games, showing which players have confirmed their presence and which have not.

## Style Guidelines:

- Primary color: Energetic, deep blue (#228AD2), evoking professionalism and team spirit, chosen for headers and interactive elements.
- Background color: A very subtle, cool light blue (#EDF5F8), creating a clean and spacious feel for content display, visually related to the primary color.
- Accent color: A vibrant, fresh aqua-green (#43CCA6) for call-to-action buttons, status indicators, and highlights, offering a refreshing contrast.
- Headline font: 'Space Grotesk' (sans-serif), for a modern, slightly technical and bold appearance, fitting for titles and key information.
- Body font: 'Inter' (sans-serif), ensuring excellent readability for all textual content, from player names to game descriptions and attendance lists.
- Utilize a consistent set of clean, minimalist vector icons that are clearly indicative of their function (e.g., user profiles, calendar, location pin, checkmark for attendance).
- A responsive, card-based layout for game listings and player profiles, prioritizing mobile readability for players confirming attendance and clear data tables for team managers.
- Subtle and fluid animations for transitions between pages, loading states, and when attendance statuses are updated, providing a smooth user experience without being distracting.
