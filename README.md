# 歡樂時光足球會 | Happy Hour Football Club

This is the official management app for Happy Hour FC, built with Next.js, Firebase, and Genkit.

## Getting Started

To run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

## Git Workflow

To push your latest changes to GitHub:

1. **Stage your changes:**
   ```bash
   git add .
   ```

2. **Commit your changes:**
   ```bash
   git commit -m "Describe your changes here"
   ```

3. **Push to GitHub:**
   ```bash
   git push origin main
   ```

## App Hosting

This app is designed to be deployed via **Firebase App Hosting**. When you push to your connected GitHub branch (usually `main`), Firebase will automatically build and deploy the new version.
